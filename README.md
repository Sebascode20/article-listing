## Overview

![screenshot](./FireShotSimple%20Article%20Listing.png)

Simple Article Listing is a lightweight responsive front-end challenge solution. The page displays a header, a separator, and three article cards with images, titles, and publish date entries (cloud computing, AI, 5G). 

The project demonstrates clean layout and typography with no JavaScript.

### What I learned

- Structuring semantic HTML with `<main>`, `<section>`, `<article>`, and proper heading hierarchy.
- Creating responsive card layouts using pure CSS flexbox and media queries.
- Using `clamp()` for responsive font sizes and ensuring accessible text scaling.
- Organizing project assets (images, icons, fonts) inside dedicated folders.

### Built with

- HTML5
- CSS3
- Flexbox layout
- Responsive breakpoints and typography

## Features

- Mobile-first responsive design.
- Article listing with image, title, and date.
- Simple and clean interface for quick reading.
- No 3rd-party frameworks or JS dependencies.
